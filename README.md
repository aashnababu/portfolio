# My Premium Portfolio 🚀

Welcome to your new portfolio! This project uses a high-end, modern design with glassmorphism and vibrant neon aesthetics.

## Quick Start
1.  Open `index.html` in any browser.
2.  Edit `index.html` to update your name, projects, and contact info.
3.  Modify `style.css` to tweak colors and typography.

## Features
-   **Neon Accents**: Modern cyan/purple glow.
-   **Glassmorphism**: Translucent, blurred backgrounds.
-   **Scroll Reveal**: Smooth animations on scroll.
-   **Custom Cursor**: Interactive mouse follower.

## Customization
Look for the `:root` variables in `style.css` to change the entire color scheme in seconds!
